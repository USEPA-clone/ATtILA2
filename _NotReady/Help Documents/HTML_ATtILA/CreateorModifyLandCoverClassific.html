Create or Modify Land Cover Classification

Land Cover Classification ››

Summary

Create custom Land Cover Classification (LCC) files or modify existing files.


The External LCC Editor will be launched separately from ArcGIS. You may close ArcGIS without disrupting the editor. A LCC file defines the classification scheme and is a required input for some tools.


Illustration


Usage

  • Allows users to modify an existing LCC XML file to suit the their specific needs. For example, the user may define a new grouped class by combining multiple individual classes (e.g., deciduous, conifer, and mixed forest into a single forest class).
  • Allows users to create a new LCC XML file for a land cover grid other than the standard datasets already included in ATtILA for ArcGIS (e.g. a local Land cover grid with a unique classification scheme).


Syntax

LaunchLccEditor ()


This  tool has no parameters.
